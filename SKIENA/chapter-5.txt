5.3
Prove by induction that there is a unique path between
any pair of vertices in a tree

answer:

let n be the number of vertices in our tree

base case:
n = 1 -> single node
n = 2 -> a pair, 2 vertices and 1 edge

inductive hypothesis
assume the property holds true for a tree of size k
and we wanan use this to prove this holds for k + 1

for any tree with k vertices, there is a unique path between any pair of vercies

inductive step
every tree with 2 or more vertices guaranteed to have 1 leaf

if we have tree iwth k + 1 vertice
remove leaf -> k + 1 - 1 = k vertices
